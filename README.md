## üöÄ Getting Started

### Prerequisites
- **For C/C++ Programming:**
  - A C/C++ compiler (GCC, MinGW, or MSVC)
  - Visual Studio Code or your preferred IDE
  - Git for version control

### Setup Options

1. **GitHub Codespaces** (Recommended for beginners):
   - Fork this repository
   - Open with GitHub Codespaces
   - Select the appropriate dev container configuration

2. **Local Development**:
   - Clone the repository: `git clone https://github.com/Nikolai-Slsr/study-code.git`
   - Install required compilers and tools (see Lab 1 for detailed instructions)
   - Open the project in your IDE

3. **DHBW Lab Computers**:
   - Follow the specific instructions in Lab 1, Section II

## üìñ How to Use This Repository

4. **Reference the solution folders** when you need guidance or want to compare approaches
